1. Introduction

Diagno is a Web Application where you can check if you have pulmonary diseases. You can upload your X-ray and get your result.

The algorithm behind is able to detect Pneumonia or SARS-COV-2 disease using Deep Learning models for classification. This Interface has a demonstration purpose of the researched algorithm trained on a big dataset in order to obtain the best result possible.

2.User Interface

2.1. Landing Page

The first page, a simple one that user will enter upon accessing the website, here we describe the purpose of the website, how to use it and other information from 'Find out more' for details .

2.2. Login Page

This is the page where the user can login or authenticate for the first time with new credentials and login in his new account.

2.3. Upload Page

This is the page where the user can upload his X-ray to get his result with the specific accuracy of classification.

3. Use cases

3.1. The user will log into an account or creates a new one

The user decides to use the page provided for his registration, here he will input his credentials for the creation of the new account.

3.2. The user gets his result

This is when the user uploads this X-ray from his personal computer and after this he receive a response with a specific accuracy.

The algorithm will return one of the following outputs: Normal if there are no signs of problems, Viral Pneumonia or Sars-Cov-2, followed by the accuracy of the classification .

4. Overview

The desired result is to obtain a classification model as accurate as possible and to make a friendly and simple to use interface for anyone who wants to try it.
